## Bluespec

Bluespec은 high-level HDL로, Verilog보다 더 높은 수준의 abstraction을 제공한다.
문법이 C/C++과 유사한 부분이 많고고, 굉장히 직관적인 형태를 가지고 있어 배우기가 쉬운 편이다. 주로 RISC-V 시뮬레이션에 사용된다.
