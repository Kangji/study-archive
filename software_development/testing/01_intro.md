# Testing

- Process of evaluating software to **gain confidence**
- Not to prove the absent of bug

## Types of Testing

### Purpose

- Functional Test
- Performance Test
- Usability Test
- Acceptance Test
- Compatibility Test

### Scope

- Unit Test
    - Focus on **success, error, edge case**
- Integration Test
    - UI Testing
- End-to-end(System) Test
    - Function Test
    - Quality Test
    - Acceptance Test (Blackbox)
    - Installation Test (Blackbox)

### Environment

- Local Test
- Instrumented Test

### Visibility

- Blackbox(what) vs. White box(how)

### Automation

- Manual(flexible) vs. Automated(fast)

## Test Strategies

- Fault Injection
- Fuzz Testing
- Mutation Testing
